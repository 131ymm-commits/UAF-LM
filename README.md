# UAF-LM: Language Model with UAF Coherence Regularization

**UAF-LM** – это эксперимент по внедрению **UAF‑когерентности (Γ)** в рекуррентные языковые модели (LSTM) для генерации текста. Регуляризация на основе γ стимулирует модель поддерживать плавность скрытых состояний, снижая переобучение и улучшая креативность.

## ✨ Особенности

- Встраивание метрики **Gamma (γ)** в функцию потерь (штраф за падение когерентности).
- Сравнение базовой LSTM и UAF-регуляризованной модели на одном корпусе.
- Генерация текста с контролем температуры.
- Простота расширения: можно заменить LSTM на Transformer, добавить γ в attention.

🧠 Как работает UAF-регуляризация?
Внутри каждого батча вычисляется γ – средняя косинусная близость между последовательными скрытыми состояниями LSTM на последних 10 тактах. Чем выше γ, тем более плавно меняются внутренние представления. Обучение минимизирует Loss = CrossEntropy + λ·(1 – γ), где λ – сила регуляризации. Это простая форма мягкого поощрения когерентности.

📈 Пример генерации
Seed: "Alice was"

Без UAF: Alice was beginning to get very tired of sitting by her sister...

С UAF: Alice was very happen “I should must be late!” the middle, she was rather out as she went into the shelves...

UAF-модель генерирует более оригинальный текст, избегая дословного копирования.

🔧 Настройка параметров
В train.py можно изменить:

--lambda_uaf – сила регуляризации (по умолчанию 0.1)

--epochs – количество эпох

--embedding_dim, --hidden_dim – размерности сети

--temperature – температура при генерации
